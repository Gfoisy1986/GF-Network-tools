# Documentation du GF‑Fortran‑SDK  
*Section `docs/` — Référence complète, guides et tutoriels*

Cette section contient **toute la documentation officielle du GF‑Fortran‑SDK**.  
Elle regroupe les explications détaillées de chaque API, module, composant et outil du SDK, ainsi que des guides pratiques et du contenu pédagogique destiné aux utilisateurs.

L’objectif est d’offrir une documentation **claire, complète et accessible**, couvrant aussi bien les bases que les fonctionnalités avancées.

---

## 🎯 Objectifs de cette section

- Centraliser **toute la documentation du SDK**  
- Expliquer chaque **API**, **module**, **composant**, **outil** et **concept**  
- Fournir des **guides pratiques**, des **tutoriels** et des **exemples commentés**  
- Servir de référence pour les développeurs débutants comme avancés  
- Documenter la vision, l’architecture et les bonnes pratiques du SDK  
- Préparer une base solide pour une documentation bilingue (FR/EN)
