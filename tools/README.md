# Outils et utilitaires du GF‑Fortran‑SDK  

*Scripts, générateurs, assistants et outils externes*

La section `tools/` regroupe tous les **outils auxiliaires** fournis avec le GF‑Fortran‑SDK.  
Ces outils facilitent l’utilisation du SDK, automatisent certaines tâches et fournissent des fonctionnalités complémentaires utiles au développement, au déploiement et aux tests.

Cette section inclut notamment des **scripts préfabriqués**, des **générateurs**, des **outils système**, ainsi que tout contenu destiné à accompagner les développeurs dans l’utilisation du SDK.

---

## 🎯 Objectifs de cette section

- Fournir des **outils pratiques** pour accompagner le développement avec le SDK  
- Automatiser des tâches courantes (certificats, builds, génération de fichiers, etc.)  
- Offrir des **scripts prêts à l’emploi** pour Windows, Linux et macOS  
- Centraliser les utilitaires nécessaires aux exemples, tests et modules du SDK  
- Servir de base pour des **tutoriels** et du contenu pédagogique  

---


## 📌 Contenu prévu

Cette section accueillera progressivement :

### ✔️ Scripts système  
- génération de certificats  
- outils de packaging  
- scripts de build multi‑plateforme  
- automatisation de tâches récurrentes  

### ✔️ Outils pour développeurs  
- générateurs de projets  
- outils de diagnostic  
- validateurs de configuration  
- convertisseurs ou extracteurs  

### ✔️ Contenu pédagogique  
- scripts utilisés dans les tutoriels  
- outils pour reproduire les exemples  
- utilitaires destinés aux guides d’apprentissage
