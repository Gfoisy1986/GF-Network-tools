## Lua (recommended scripting language)

Lua is a lightweight, embeddable scripting language used to automate builds,
generate code, and extend the GF-Fortran-SDK toolchain.

- License: MIT (very permissive)
- Platforms: Windows, Linux, macOS, Raspberry Pi, BSD, etc.
- Perfect for scripting around GCC, gfortran, and NASM.
- Easy to embed in C applications.


Tu pourras mettre ça dans tools/lua/examples/hello.lua :

```

print("Lua is running inside GF-Fortran-SDK!")

```

```bash

lua tools/lua/examples/hello.lua

```
