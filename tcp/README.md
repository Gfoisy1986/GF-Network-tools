# TCP Server & Client

Ce module fournit une implémentation simple et propre d’un serveur et d’un client TCP.  
Il repose sur les wrappers C et les modules bas niveau du dossier `net/`.

## Contenu
- `server/` : Serveur TCP minimal
- `client/` : Client TCP minimal

## Fonctionnalités
- Connexion TCP simple
- Envoi et réception de chaînes
- Gestion basique des clients
- API claire et facile à utiliser

## Objectif
Offrir une base TCP propre pour construire des protocoles personnalisés, des agents, des services internes ou des tests réseau.
